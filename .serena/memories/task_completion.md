# Task Completion Checklist

When completing a task in this project:

1. **Build Verification**
   - Run `npm run build` to ensure TypeScript compiles without errors

2. **Script Verification**
   - Test scripts with `npx ts-node scripts/<script>.ts`
   - Verify expected output

3. **Documentation**
   - Update relevant docs in `docs/` folder
   - Update STATUS.md with progress

4. **Version Control**
   - Stage relevant changes
   - Commit with descriptive message

## Phase-Specific Checks
- **Phase 1:** Device detection and raw input capture work
- **Phase 2:** Protocol parsing functions tested
- **Phase 3:** Action handlers execute correctly
- **Phase 4:** Configuration saves/loads properly
- **Phase 5:** Electron app builds and runs
- **Phase 6:** Integrations connect successfully
- **Phase 7:** Installer creates working package
